# EcoGestor

Sistema **EcoGestor** desenvolvido para apoiar a gestão sustentável por meio do acompanhamento de dados, automações e visualizações que auxiliam na tomada de decisão.

## 📌 Proposta do Projeto
O EcoGestor tem como objetivo central facilitar o gerenciamento de informações relacionadas à sustentabilidade, oferecendo uma plataforma prática para organização, análise e acompanhamento de indicadores.

## 🚀 Tecnologias Utilizadas
Com base na composição do repositório:

- **JavaScript** (56.6%)
- **Python** (31.0%)
- **Java** (6.5%)
- **HTML** (4.1%)
- **CSS** (1.8%)

## ▶️ Como Executar o Projeto Localmente

> **Pré-requisitos** (ajuste conforme seu projeto):
- [Node.js](https://nodejs.org/) instalado
- [Python 3](https://www.python.org/) instalado
- [Java JDK](https://www.oracle.com/java/technologies/downloads/) instalado (se houver serviços Java)
- Git instalado

### 1. Clonar o repositório
```bash
git clone https://github.com/Jp0Martins/EcoGestor.git
cd EcoGestor
```

### 2. Configurar e executar a parte JavaScript (frontend ou API Node)
```bash
# entre na pasta JS (ex.: frontend ou backend)
cd <pasta-javascript>
npm install
npm run dev
```

### 3. Configurar e executar a parte Python
```bash
# volte para a raiz se necessário
cd <pasta-python>
python -m venv venv
# Linux/macOS:
source venv/bin/activate
# Windows:
venv\Scripts\activate

pip install -r requirements.txt
python main.py
```

### 4. Executar a parte Java (se aplicável)
```bash
cd <pasta-java>
# Exemplo com Maven
mvn clean install
mvn spring-boot:run
```
